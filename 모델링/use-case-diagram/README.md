## 📖 UML이란?

### UML

- 클래스 다이어그램
- 유즈 케이스(Use Case)
    - 사용자 또는 관리자 관점에서 바라본 시스템 or 웹 사이트 내용에 대한 '시나리오'의 집합으로 볼 수 있다.
    - 시스템 개발 과정의 모든 단계를 끌어낸다.
    - 사용자 시각에서 소프트웨어 범위와 기능을 설명하는 설계 표기법이라고도 한다.
